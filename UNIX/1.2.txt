计算机语言：机器、汇编、高级

1. 预处理cpp： .i 预处理指令、删除注释
2. gcc： .s     编译为汇编文件   
3. 进行汇编 .o
4. 绑定链接： .out

过程：
1. 预处理过程
    gcc -E hello.c -o hello.i
2. 编译 ：编译为汇编语言
    gcc -S hello.i -o hello.s
3. 汇编 ：汇编语言转换为机器指令
    gcc -c hello.s 
4. 链接 ：将目标文件和运行时文件、库函数合成为可执行文件
    gcc hello.o

为什么#include<stdio.h> 没有 ;
    给预处理器的指令
    #开头都是预处理指令

常见的预处理指令
    1. 宏定义
    2. 条件编译
    3. 文件的包含

二进制文件分为两种：一种是可执行的，一种是不可执行的

ls -l 文件      //列出文件的权限
chmod a+x 文件  //增加可执行权限
file 文件       //列出文件的性质

GCC参数小结：
-E 预处理
-S 将源文件或预处理后的程序编译为汇编语言
-c 编译为目标文件
-o 输出文件

文件名后缀的小结：
.c 源文件
.i 预处理后的文件
.s 汇编语言文件
.h 头文件
.o 目标文件
