# shell

## 管道

### 本质

管道是内核中的一块缓冲区(内存),有两个文件描述符

* pipefd[0]:读端(read end)
* pipefd[1]:写端(write end)

```c
内核空间
┌─────────────────────────┐
│     管道缓冲区           │
│  ┌───┬───┬───┬───┐      │
│  │ d │ a │ t │ a │      │
│  └───┴───┴───┴───┘      │
│    ↑           ↓        │
│   写端        读端       │
└─────────────────────────┘
     ↓           ↓
  pipefd[1]   pipefd[0]
   (写描述符)   (读描述符)
```

### 工作机制

#### 创建管道

```c
#include <stdio.h>
#include <unistd.h>

int main() {
    int fd[2];
    
    if (pipe(fd) == -1) {
        perror("pipe");
        return 1;
    }
    
    printf("读端 fd[0] = %d\n", fd[0]);  // 通常是 3
    printf("写端 fd[1] = %d\n", fd[1]);  // 通常是 4
    
    // 现在可以像普通文件一样读写
    write(fd[1], "Hello", 5);
    
    char buffer[10];
    read(fd[0], buffer, 5);
    buffer[5] = '\0';
    printf("读到: %s\n", buffer);  // 输出: Hello
    
    close(fd[0]);
    close(fd[1]);
    
    return 0;
}
```

#### 内核数据结构

```c
// 内核中的管道结构（简化）
struct pipe_buffer {
    struct page *page;      // 内存页
    unsigned int offset;    // 偏移量
    unsigned int len;       // 数据长度
};

struct pipe_inode_info {
    struct pipe_buffer *bufs;  // 环形缓冲区
    unsigned int nrbufs;       // 已使用的缓冲区数
    unsigned int curbuf;       // 当前缓冲区
    unsigned int buffers;      // 总缓冲区数
    wait_queue_head_t wait;    // 等待队列
};
```

### 行为规则

#### **读端行为**

| 情况             | 行为     | 返回值           |
| ---------------- | -------- | ---------------- |
| 管道有数据       | 读取数据 | 实际读取的字节数 |
| 管道空，写端打开 | 阻塞等待 | 阻塞             |
| 管道空，写端关闭 | 立即返回 | 0 (EOF)          |

#### **写端行为**

| 情况             | 行为         | 返回值         |
| ---------------- | ------------ | -------------- |
| 管道有空间       | 写入数据     | 实际写入字节数 |
| 管道满，读端打开 | 阻塞等待     | 阻塞           |
| 管道满，读端关闭 | 收到 SIGPIPE | -1 (EPIPE)     |

### 父子进程通信

```c
父进程                      子进程
   │                          │
   ├─ pipe(fd) ───────────────┤
   │                          │
   │  fd[0] (读端)            │  fd[0] (读端)
   │  fd[1] (写端)            │  fd[1] (写端)
   │                          │
   ├─ close(fd[0])            ├─ close(fd[1])
   │  (关闭读端)               │  (关闭写端)
   │                          │
   │  write(fd[1], data)      │  read(fd[0], buf)
   │       ↓                  │       ↑
   │   [内核缓冲区] ──────────┘
   │
```

### 管道 vs 其他IPC

| 特性            | 管道   | 消息队列 | 共享内存 | Socket      |
| --------------- | ------ | -------- | -------- | ----------- |
| **数据拷贝**    | 2次    | 2次      | 0次      | 2次         |
| **无连接通信**  | 否     | 是       | 否       | 是          |
| **内核维护**    | 是     | 是       | 否       | 是          |
| **字节流/消息** | 字节流 | 消息     | 字节流   | 字节流/消息 |
| **单向/双向**   | 单向   | 双向     | 双向     | 双向        |
| **速度**        | 慢     | 中       | 快       | 中          |

### 简要总结

| 概念         | 说明                   |
| ------------ | ---------------------- |
| **本质**     | 内核中的环形缓冲区     |
| **数据流向** | 单向（一端写，一端读） |
| **通信方式** | 字节流                 |
| **生命周期** | 跟随文件描述符         |
| **阻塞行为** | 默认阻塞，可设为非阻塞 |
| **原子性**   | ≤ PIPE_BUF 时原子      |
| **适用场景** | 父子/兄弟进程通信      |

## pipe()

```c
#include <unistd.h>
int pipe(int pipefd[2]);
```

创建一个单向数据通道(管道),用于进程间通信



成功返回0

失败返回-1



特点

```c
int fd[2];
pipe(fd);

// fd[0] - 读端（read end）
// fd[1] - 写端（write end）

// 数据流向：fd[1] → 管道缓冲区 → fd[0]
```



内核实现

```c
用户空间                    内核空间                    用户空间
─────────                  ─────────                  ─────────
    │                           │                           │
    ├─ write(fd[1], data) ───→  │                           │
    │                           ├─ 管道缓冲区                │
    │                           │  ┌────┬────┬────┐         │
    │                           │  │ H  │ e  │ l  │ ...     │
    │                           │  └────┴────┴────┘         │
    │                           │                           │
    │                           ├─ read(fd[0], buf) ←───────┤
    │                           │                           │
```

## strchr()

```c
#include <string.h>
char *strchr(const char *s, int c);
```

在字符串s中查询c第一次出现的位置



* 返回值:

  	* 成功:返回指向第一次出现位置的指针

  	* 失败:返回NULL
