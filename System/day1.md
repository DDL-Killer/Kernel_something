# 打开电源

## 计算模型

### x86 PC

1. 开机时CPU处于实模式--(和保护模式对应,实模式的寻址CS:IP(CS左移4位+IP),和保护模式不一样)
2. 开机时,CS=0xFFFF;IP=0x0000
3. 寻址0xFFFF0(ROM BIOS映射区)

4. 检查RAM,键盘,显示器,软硬磁盘
5. 将磁盘0磁道0扇区读入0x7c00处(引导扇区)
6. 设置cs=0x07c0,ip=0x0000

# 引导扇区--0x7c00处存放的代码

## 从磁盘引导扇区读入的那512个字节

* 是启动设备的第一个扇区
* 启动设备信息被设置在CMOS中
* 第一个扇区上存放着开机后执行的第一段可以控制的程序
* 操作系统的故事由此开始...
